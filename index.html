PostAPP

Ferramenta para publicação de vídeos no TikTok via API oficial.

O PostAPP integra-se com o TikTok for Developers para permitir a publicação de conteúdo de vídeo diretamente na conta autorizada do usuário, usando os escopos video.publish e video.upload.
